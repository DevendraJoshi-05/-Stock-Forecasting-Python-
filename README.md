# Stock Price Prediction Web App

A Machine Learning web application to predict and analyze stock prices 
using Deep Learning models (LSTM & GRU).

## Features
- Stock price forecasting using LSTM and GRU neural networks
- Technical analysis of stock market data
- Fundamental analysis support
- Correlation analysis between stocks
- Interactive web interface using Flask

## Tech Stack
- Python, Flask
- LSTM, GRU, Keras, TensorFlow
- Jupyter Notebook
- Matplotlib, Seaborn

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run the app: python app.py
3. Open browser: http://localhost:5000
